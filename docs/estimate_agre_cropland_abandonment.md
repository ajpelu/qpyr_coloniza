# Cropland Abandonment Age

Aerial photography were taken from *Fototeca Digital* (Spanish National Geographic Institute, IGN). To select the aerial photograms of our sutdy areas, we used the online [viewer](http://fototeca.cnig.es/) of IGN. 

In the next table we list the photograms used. For each photogram we detailed: 

 * `vuelo`: Flight name
 * `Referencia`: original file reference
 * `fotograma`: name of the photogram
 * `cultivo`: study cropland 
 * `fecha_fot`: original date of the photogram
 * `year`: year of the photogram
 * `file`: name of the file. All files are stored in `./images/age_crops/photograms/`
 * `escalaVuelo`: original scale 
 * `escalaFile`: scale of the cropland view
